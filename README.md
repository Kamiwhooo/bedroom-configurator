# 🛏 3D Bedroom Configurator

An interactive 3D bedroom configurator built with **Next.js 14**, **React Three Fiber**, **Three.js**, and **Tailwind CSS**.

## Features
- 🏠 Realistic 3D bedroom with proper proportions
- 🎨 Click any object to change its color (HEX, RGB, color wheel)
- 💾 Save/load room configurations as JSON
- 📷 Export room screenshot as PNG
- 🌙 Day/Night lighting toggle
- 📱 Mobile-first with touch controls
- 🖥 Desktop sidebar layout

## Run locally
```bash
npm install
npm run dev
```
